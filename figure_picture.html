Figure Elements

The figure element is used to encapsulate a self-contained content such as images, diagrams, illustrations, code snippets, etc. It provides a semantic way to group together content and its related caption or legend. The purpose of the figcaption element is to provide a caption or description for the content encapsulated within the figure element. This helps to associate the caption directly with the content it describes, enhancing accessibility and improving the structure and semantics of the HTML document.

Art Direction - Picture elements

When the browser window width is less than or equal to 640 pixels, the image-small.jpg will be displayed.



When the browser window width is greater than 640 pixels but less than or equal to 1024 pixels, the image-medium.jpg will be displayed.



When the browser window width is greater than 1024 pixels, the image-large.jpg will be displayed.
