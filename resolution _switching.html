Resolution Switching - Modern Img Tags

The srcset attribute allows us to provide multiple image sources with different resolutions. The sizes attribute specifies the display size of the image based on media conditions, such as viewport width. In this example, the browser will choose the appropriate image source from srcset based on the available viewport width and the conditions specified in sizes.
